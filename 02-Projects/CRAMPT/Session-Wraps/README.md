# CRAMPT Session Wraps

This directory contains detailed session wrap summaries for each major development session.

## Purpose

Session wraps provide:
- Detailed record of work completed
- Issues encountered and resolutions
- Technical decisions and rationale
- Testing outcomes
- Regulatory compliance verification
- Performance metrics
- Next steps and handoff information

## Naming Convention

Session wrap files should be named:
```
YYYY-MM-DD-feature-name.md
```

Examples:
- `2025-10-20-project-setup.md`
- `2025-10-22-risk-register.md`
- `2025-10-25-controls-library.md`

## Template

Use the template at `../session-wrap-template.md` for all session wraps.

## Retention

Session wraps serve as project audit trail and should be retained for the lifetime of the project.
