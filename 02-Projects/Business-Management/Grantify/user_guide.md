# MIFIDPRU Web Application User Guide

## Introduction

The MIFIDPRU Web Application is a comprehensive tool designed to help investment firms comply with the FCA's MIFIDPRU regulations and the Internal Capital Adequacy and Risk Assessment (ICARA) process. This application provides a user-friendly interface for managing all aspects of MIFIDPRU compliance, including capital requirements, risk assessment, liquidity management, stress testing, and regulatory reporting.

## Getting Started

### Accessing the Application

The MIFIDPRU Web Application can be accessed through your web browser at the URL provided by your administrator. The application is compatible with all modern web browsers including Chrome, Firefox, Safari, and Edge.

### Navigation

The application features a tabbed interface for easy navigation between different sections:

- **Dashboard**: Overview of key metrics and compliance status
- **Financial Data**: Input and manage core financial metrics
- **Risk Assessment**: Identify and assess risks using the impact/probability matrix
- **K-Factors**: Calculate K-factor requirements with FCA coefficients
- **ICARA Calculations**: Calculate and monitor ICARA requirements
- **Wind-Down**: Plan and calculate costs for an orderly wind-down
- **Liquidity Risk**: Assess and manage liquidity requirements
- **Stress Testing**: Create and run stress scenarios
- **Reverse Stress Testing**: Identify scenarios that would cause business failure
- **Historical Data**: Track and analyze trends in key metrics
- **Reporting**: Generate regulatory and internal reports
- **Compliance Dashboard**: Monitor overall compliance status
- **MIFIDPRU Settings**: Configure application settings

## Key Features

### Dashboard

The Dashboard provides an at-a-glance view of your firm's MIFIDPRU compliance status, including:

- Key financial metrics
- Capital requirements
- Liquidity status
- Risk assessment summary
- Recent changes and updates

### Financial Data

The Financial Data section allows you to input and manage your firm's core financial metrics:

1. Enter your firm's name and FCA FRN
2. Input financial data such as own funds, liquid assets, annual revenue, and expenses
3. Save changes to update calculations throughout the application

### Risk Assessment

The Risk Assessment section helps you identify and assess risks using the FCA's impact/probability matrix:

1. View the risk matrix showing impact and probability scores
2. Add new risks with impact and probability ratings
3. View calculated risk scores and additional capital requirements
4. Monitor total risk capital

### K-Factors

The K-Factors section calculates capital requirements based on the FCA's K-factor approach:

1. View existing K-factors with their values and calculated requirements
2. Add new K-factors from the available list
3. Specify whether values are for cash or derivatives
4. Monitor the total K-factor requirement (KFR)

### ICARA Calculations

The ICARA Calculations section determines your overall capital requirement:

1. View Pillar 1 components (PMR, FOR, KFR)
2. Input Pillar 2 components (Risk Capital)
3. View Wind-Down cost
4. Compare Own Funds against the ICARA requirement
5. Monitor surplus or deficit

### Wind-Down Planning

The Wind-Down section helps you plan for an orderly wind-down of your business:

1. View existing cost categories with monthly costs and wind-down periods
2. Add new cost categories
3. Monitor total wind-down cost
4. Track maximum wind-down period

### Liquidity Risk Assessment

The Liquidity Risk section helps you manage liquidity requirements:

1. View calculated LATR, OFLATR, and BLATR
2. Compare liquid assets against requirements
3. Monitor compliance with liquidity thresholds

### Stress Testing

The Stress Testing section allows you to assess your firm's resilience to stress:

1. View predefined stress scenarios (Base Case, Mild, Severe, Extreme)
2. Create custom stress scenarios
3. Specify impact percentages for own funds, liquidity, and K-factors
4. Run stress tests to see the impact on your financial position

### Reverse Stress Testing

The Reverse Stress Testing section helps you identify scenarios that would cause business failure:

1. Define failure thresholds
2. Describe scenarios that could breach these thresholds
3. Identify key risk drivers
4. Document potential management actions

### Historical Data

The Historical Data section allows you to track and analyze trends:

1. Input historical data for key metrics
2. View trend charts
3. Filter data by metric type
4. Monitor changes over time

### Reporting

The Reporting section generates regulatory and internal reports:

1. Generate FCA regulatory reports (MIF001-MIF007)
2. Preview report content
3. Download reports in various formats (XML, PDF, Excel)
4. View submission instructions

### Compliance Dashboard

The Compliance Dashboard provides a comprehensive view of your compliance status:

1. View overall compliance status
2. Identify compliance issues with severity levels
3. Monitor capital requirements and surplus/deficit
4. Track liquidity requirements compliance

### MIFIDPRU Settings

The MIFIDPRU Settings section allows you to configure the application:

1. Update firm details
2. Configure MIFIDPRU parameters
3. View K-factor coefficients
4. Enable/disable features
5. Set reporting frequency

## Best Practices

### Regular Updates

- Update financial data at least quarterly
- Review risk assessments whenever business conditions change
- Run stress tests at least annually
- Update wind-down planning when cost structures change

### Compliance Monitoring

- Check the Compliance Dashboard regularly
- Address any compliance issues promptly
- Maintain adequate capital and liquidity buffers
- Document all compliance activities

### Reporting

- Generate and review reports before submission deadlines
- Keep copies of all submitted reports
- Follow FCA submission instructions carefully
- Update historical data to track trends

## Troubleshooting

### Common Issues

- **Calculation errors**: Ensure all input data is correct and complete
- **Missing data**: Check that all required fields are filled in
- **Report generation failures**: Verify that all necessary data is available
- **Login issues**: Contact your administrator to reset credentials

For technical support, please contact your system administrator or the support team at support@mifidpru-app.com.

## Glossary

- **PMR**: Permanent Minimum Requirement
- **FOR**: Fixed Overhead Requirement
- **KFR**: K-Factor Requirement
- **ICARA**: Internal Capital Adequacy and Risk Assessment
- **LATR**: Liquid Asset Threshold Requirement
- **OFLATR**: Own Funds Liquid Asset Threshold Requirement
- **BLATR**: Basic Liquid Assets Threshold Requirement
- **FCA**: Financial Conduct Authority
- **FRN**: Firm Reference Number
