# MIFIDPRU Application Licensing Tier Structure

## By Firm Size

### Tier 1: Small Firms
- **Assets Under Management (AUM)**: Up to £100 million
- **Staff**: Up to 10 employees
- **Client base**: Up to 100 clients
- **Features**: Core MIFIDPRU compliance functionality, basic reporting

### Tier 2: Medium Firms
- **AUM**: £100 million to £1 billion
- **Staff**: 11-50 employees
- **Client base**: 101-500 clients
- **Features**: All Tier 1 features + advanced stress testing, enhanced reporting, API access

### Tier 3: Large Firms
- **AUM**: Over £1 billion
- **Staff**: 51+ employees
- **Client base**: 500+ clients
- **Features**: All Tier 2 features + multi-entity support, custom risk models, white-labeling options

## By Subscription Duration

### Monthly Access
- Highest per-month cost (premium for flexibility)
- No commitment required
- Ideal for firms with seasonal needs or testing the solution
- **Small**: £500/month
- **Medium**: £1,200/month
- **Large**: £2,500/month

### Quarterly Access (60-90 days)
- 10% discount compared to monthly rate
- Moderate commitment
- Aligns with quarterly reporting cycles
- **Small**: £1,350/quarter (£450/month equivalent)
- **Medium**: £3,240/quarter (£1,080/month equivalent)
- **Large**: £6,750/quarter (£2,250/month equivalent)

### Annual Access
- 20% discount compared to monthly rate
- Annual commitment
- Most cost-effective for long-term use
- **Small**: £4,800/year (£400/month equivalent)
- **Medium**: £11,520/year (£960/month equivalent)
- **Large**: £24,000/year (£2,000/month equivalent)

## Feature-Based Premium Tiers

### Standard Edition
- Base pricing as outlined above
- Core MIFIDPRU compliance functionality
- Standard support (email, business hours)

### Professional Edition
- 25% premium over Standard pricing
- Additional features: Advanced historical data analysis, custom dashboards, priority support
- Enhanced API access with higher rate limits
- Support: Email + phone during extended hours

### Enterprise Edition
- 50% premium over Standard pricing
- All Professional features + multi-entity support, custom integrations
- Dedicated account manager
- Custom development hours included (10 hours/quarter)
- Support: 24/7 priority support with SLAs

## Add-On Options

### Implementation Package
- One-time fee: £2,000 (Small), £5,000 (Medium), £10,000 (Large)
- Includes data migration, configuration, and training

### Regulatory Updates Service
- Annual fee: £1,000 (Small), £2,500 (Medium), £5,000 (Large)
- Guaranteed updates for regulatory changes within 30 days

### Custom Development
- Hourly rate: £150/hour
- Minimum engagement: 10 hours

## License Key Structure

Each license key will encode the following information:
- Firm size tier (1, 2, or 3)
- Edition (Standard, Professional, Enterprise)
- Subscription duration (Monthly, Quarterly, Annual)
- Expiration date
- Licensed features
- Maximum user count

## License Enforcement

The license management system will enforce:
- Access to features based on tier and edition
- User count limitations
- Expiration dates with grace periods
- API rate limits based on tier
